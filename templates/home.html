What is a Generative Adversarial Network?


If you were given 1000 images, could you then make one new image that could fit into the original 1000? That is the basic theory behind GANs. Given a group of data, can we create a new instance that can fit into that group. This is done with neural networks that work together to find the distribution of the data. Once the distribution is known, it is "easy" to make a new point that falls in the distribution.

How does a basic GAN work?


Latent Space

This is the input to the GAN. It is really just a list of random numbers between 0-1 or -1-1. This space esentially gives the GAN blank space to pull from. Each of the numbers affect the output. For example, in image creation, one number could change the hair color, while another could change the overall hue. This list is given to the Generative Model.

Generative Model

When making new images, only the generative network is used from the GAN. This network has been trained on a specific set of data, and using the latent space, is able to create new content that matches the training data.

Output

In the examples that I made, the output is a 32 x 32 array. This means a 1 X 100 list was transformed to create a new array of 32 x 32 numbers. We can then change those number to pixels and make an image.

How does a Cycle GAN work?


Two Sides

The purpose of a cycle GAN is to transfer information from one area to another. In my examples, real images we converted to a specific painter's style. In a cycle GAN, there are really two GANs working together. One turns an image to look like a painting and the other turns paintings to look like images. The logic behind a this type of GAN is that an image should convert back to itself by running through both GANs.

This is easy to comprehend when thinking about language translation. If you translate a sentance from Spanish to English, it should translate back to the same thing from English to Spanish.

Choose One Generative Model

In the examples above, only one of the two GANs is used; the one that made photos look like paintings. This is given an image from a user. The cycle GANs were trained on 256 x 256 images, so when an image is choosen, it is resized to that to be fed into the model.

Output

The output is a 256 x 256 image that has been transformed into the chosen style.
